std::shared_ptr<textLayout_c> STLL::codepointAttributes::inlay

The inlay is a layout that will be placed instead of an actual glyph. If you define the inlay, the font information will be ignored. How the inlay interacts with line breaks depends on the character that is given as the placeholder. Linebreaks are done as if there was the given placeholder character and not the inlay.

The vertical alignment of the inlay is controlled by baseline_shift. If baseline_shift is 0 the inlay is placed on the baseline.
